# Control de Asistencia Escolar

## Elevator Pitch
Aplicación web que permite a los docentes registrar la asistencia de sus alumnos de forma digital y organizada, facilitando la generación de reportes y mejorando el control escolar.

## Tecnologías
- MongoDB
- Express.js
- React
- Node.js
- Git & GitHub

## Objetivo
Mejorar la eficiencia del registro de asistencias y reducir el uso de papel en instituciones educativas.
